# 应届生求职助手 —— JD 分析 + 简历改写

> 一个基于 LLM 的工具，解析岗位描述并生成简历修改建议。
> 这是一个**作业级别的原型**，不是产品。

## 项目简介

本工具包含两个模块：
- **JD 分析**：从岗位描述中提取职责、技能、加分项、行业分类，输出结构化数据和关键词关系图。
- **简历改写**：输入 JD 和你原有的简历，自动生成按标签分类的修改建议 + 参考话术。

技术栈：Python + Streamlit + 大模型 API（云端）

## 为什么做这个项目？

因为我没有实习经历，又幻想直接当 AI 产品经理。所以做了一个工具来“证明”自己有产品能力。但现实是：**这个工具目前只有你自己在用，还不知道能不能跑通**。
